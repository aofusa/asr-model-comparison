# Backlog

このファイルでは、バグ対応と機能開発の一覧および進行状況を管理する。

- `docs/backlog/bugs/`: 検知したバグの詳細ドキュメントを保存する。
- `docs/backlog/features/`: 今後の機能開発の詳細ドキュメントを保存する。
- バグ対応や機能開発を行ったら、この `backlog.md` の状態を更新する。
