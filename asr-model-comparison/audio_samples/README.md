# テスト用音声サンプル (Test Audio Samples)

このディレクトリには、ASRモデル比較に使用する**同一内容の多言語音声ファイル**を配置します。

## 必要な音声ファイル

各言語で**内容・長さ・話者ができるだけ近い**短い音声（5〜30秒程度）を3〜5セット用意してください。

推奨ファイル名例:

- `ja_01.wav` — 日本語（例: 天気予報やニュース風の短い文章）
- `en_01.wav` — 英語（同じ内容の翻訳）
- `zh_01.wav` — 中国語（同じ内容の翻訳）

### 推奨内容例（短いパッセージ）

**日本語例**:
「本日の東京の天気は晴れ、最高気温は23度です。夕方から少し雲が増えるでしょう。」

**英語例**:
"Today's weather in Tokyo is sunny with a high of 23 degrees. Clouds will increase in the evening."

**中国語例**:
"今天东京的天气晴朗，最高气温23度。傍晚时分云量会略有增加。"

## ファイル形式

- 推奨: 16kHz モノラル WAV（PCM）
- 許容: MP3, M4A, FLAC（pydub / torchaudio が自動変換）
- サンプリングレート: 16kHz または 48kHz 推奨（Whisper等が内部で16kHzにリサンプリング）

## 入手方法

1. 自分で録音（高品質マイク、静かな環境）
2. 公共ドメインの音声コーパスから抜粋（Common Voice, JSUT, などライセンス確認必須）
3. 合成音声（TTS）で生成（ただし実世界性能評価には実録音を推奨）

## 注意事項

- **著作権**: 必ずライセンスが明確なものを使用。商用利用不可のものは避ける。
- **個人情報**: 実在の人物名・電話番号・住所などは絶対に含めない。
- **ファイルサイズ**: 各ファイル 1MB 未満を目安に（git-lfs 検討可）。
- 大きなファイルは `.gitignore` により追跡されません。必要に応じて `git lfs track` を使用。
